# Niveles de Cumplimiento

| Nivel | Rango | Descripción |
|-------|-------|-------------|
| SHADOW | 0-19 | Sin identidad verificada |
| BASIC | 20-39 | Identidad creada, delegación incompleta |
| VERIFIED | 40-69 | Delegación válida, vinculación territorial confirmada |
| TRUSTED | 70-89 | Historial consistente, staking activo |
| SOVEREIGN | 90-100 | Cumplimiento completo, nivel enterprise |
