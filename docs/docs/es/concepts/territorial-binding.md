# Vinculación Territorial

Los agentes se vinculan a jurisdicciones geográficas mediante celdas hexagonales H3.

[H3](https://h3geo.org/) es el sistema de cuadrícula hexagonal jerárquica de Uber. GNS-AIP usa celdas de resolución 10 (~15m²).

```
Celda H3 "8a2a1072b59ffff"
    → Coordenadas: 41.89°N, 12.49°E
    → Ciudad: Roma, Italia
    → Jurisdicción: UE (GDPR, AI Act UE, eIDAS 2.0)
```

Las coordenadas GPS raw nunca se almacenan. Todas las ubicaciones se cuantizan en celdas H3.
