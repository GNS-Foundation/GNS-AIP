# SDK TypeScript

`@gns-aip/sdk` es el SDK core de TypeScript para GNS-AIP. Todas las integraciones TypeScript se basan en este paquete.

```bash
npm install @gns-aip/sdk
```

- **GNSAgentSDK** — Cliente HTTP para todos los endpoints
- **DelegationChain** — Recorre y verifica cadenas de delegación
- **EscalationPolicy** — Evalúa cuándo escalar a supervisión humana
- **ComplianceTier** — Enum de 5 niveles con operadores de comparación

128 tests en todos los módulos.
