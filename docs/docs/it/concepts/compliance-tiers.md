# Livelli di Conformità

GNS-AIP utilizza un modello a 5 livelli che valuta gli agenti da 0 a 100 su quattro dimensioni.

## Livelli

| Livello | Range | Descrizione |
|---------|-------|-------------|
| SHADOW | 0-19 | Nessuna identità verificata |
| BASIC | 20-39 | Identità creata, delega incompleta |
| VERIFIED | 40-69 | Delega valida, vincolo territoriale confermato |
| TRUSTED | 70-89 | Cronologia coerente, staking attivo |
| SOVEREIGN | 90-100 | Conformità completa, livello enterprise |
